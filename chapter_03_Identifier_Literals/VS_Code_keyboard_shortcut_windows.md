## Tips

- **Command Palette**: `Ctrl + Shift + P` is your best friend! You can access almost any command from here.
- **Keyboard Shortcuts Editor**: Press `Ctrl + K` then `Ctrl + S` to view and customize all shortcuts.
- **Chord Keybindings**: Some shortcuts use multiple key combinations (e.g., `Ctrl + K` then `Ctrl + S`). Release the first chord before pressing the second.
- **Context Awareness**: Some shortcuts behave differently depending on context (e.g., terminal vs. editor).
