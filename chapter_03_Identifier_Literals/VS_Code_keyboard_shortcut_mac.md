## Tips

- **Command Palette**: `Cmd + Shift + P` is your best friend! You can access almost any command from here.
- **Keyboard Shortcuts Editor**: Press `Cmd + K` then `Cmd + S` to view and customize all shortcuts.
- **Chord Keybindings**: Some shortcuts use multiple key combinations (e.g., `Cmd + K` then `Cmd + S`). Release the first chord before pressing the second.
- **Context Awareness**: Some shortcuts behave differently depending on context (e.g., terminal vs. editor).
- **macOS Differences**: Mac uses `Cmd` (Command) instead of `Ctrl` for most shortcuts, and `Option` instead of `Alt`.
